# 3. Palindrome Check and Frequency Count

**Difficulty:** Medium  
**Topics:** `String`, character frequency, loops, conditionals, primitive arrays

## Problem Statement
Write a Java program that reads a string and does the following:

1. Checks whether the string is a palindrome.
2. Prints the frequency of each character present in the string.

## Rules
- Do **not** use built-in `String` methods to solve the problem.
- Do **not** use any ready-made collection like `HashMap` or `TreeMap`.
- Use only arrays, loops, primitive variables, and the `String` object for input.
- Ignore spaces while checking palindrome.

## Input
A single line string.

## Output
- Print whether the string is a palindrome.
- Print the frequency of each character.

## Example
**Input**
```text
nurses run
```

**Output**
```text
Palindrome: Yes
Character Frequencies:
n -> 2
u -> 2
r -> 2
s -> 2
e -> 1
```

## Extra Requirement
The palindrome check must be case-insensitive, but you may not use built-in case-conversion methods.
