## 🧩 Problem 1 – Smart Data Analyzer

### 🎯 Objective
Analyze user data and calculate a health metric.

### 🧾 Input
- Name  
- Age  
- Height (in cm)  
- Weight (in kg)  

### ⚙️ Tasks
- Convert height → meters  
- Calculate BMI  
- Display:
  - Name  
  - Height (meters)  
  - BMI  
  - Rounded BMI  

### 🧪 Test Case
```

Input:
Rahul
22
175
70

Output:
Hello Rahul
Height in meters: 1.75
BMI: 22.857142857
Rounded BMI: 22

```
