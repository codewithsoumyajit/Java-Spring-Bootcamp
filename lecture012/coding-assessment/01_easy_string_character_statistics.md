# 1. Character Statistics in a String

**Difficulty:** Easy  
**Topics:** `String`, `char`, primitive data types, loops, conditionals

## Problem Statement
Write a Java program that reads a full line of text and counts:

- total number of alphabets
- total number of vowels
- total number of consonants
- total number of digits
- total number of spaces
- total number of special characters

## Rules
- Do **not** use any built-in `String` methods for solving the problem.
- Use only loops, conditions, primitive variables, and the `String` object for input storage.
- You may not use methods such as `substring()`, `toLowerCase()`, `toUpperCase()`, `split()`, `replace()`, `contains()`, `indexOf()`, or similar helpers.

## Input
A single line of text.

## Output
Print the counts in a clear format.

## Example
**Input**
```text
Hello World 123!
```

**Output**
```text
Alphabets: 10
Vowels: 3
Consonants: 7
Digits: 3
Spaces: 2
Special Characters: 1
```

## Extra Requirement
Treat both uppercase and lowercase letters as alphabets. Count vowels manually without converting the whole string using built-in methods.
