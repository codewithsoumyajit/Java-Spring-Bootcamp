# 2. Reverse a String Without Built-in Methods

**Difficulty:** Easy-Medium  
**Topics:** `String`, `char`, arrays, loops, primitive data types

## Problem Statement
Write a Java program that reads a string and prints its reverse.

## Rules
- Do **not** use any built-in `String` methods for reversing.
- Do **not** use `StringBuilder.reverse()`, `substring()`, `toCharArray()`, or any similar helper.
- Solve it manually using loops and primitive variables.

## Input
A single line string.

## Output
The reversed string.

## Example
**Input**
```text
Programming
```

**Output**
```text
gnimmargorP
```

## Extra Requirement
After printing the reversed string, also print whether the original string is a palindrome or not, using manual comparison only.
